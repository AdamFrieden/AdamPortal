### **Overview**

**Goals:**
* Learn new stuff
* Have something out in the wild to reference
* Starting point for other projects

**Technology:**
- React (+Router)
- Typescript
- Vite
- npm
- GitHub (+Actions)
- Material UI
- AWS S3 (+Cloudfront)

Notably did not use:
- Next.js
- Vercel
- CRA

### **How to Run**

Run local dev server
`npm run dev`
